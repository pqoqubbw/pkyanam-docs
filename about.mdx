I'm Preetham Kyanam, a software engineer based in Ashburn, Virginia. I graduated from Virginia Tech in May 2023 with a B.S. in Computer Engineering, concentrating in Controls, Robotics and Autonomy.

I work across full stack development, AI agent infrastructure, and developer tooling. Professionally, I build internal web and mobile applications at Air Culinaire Worldwide and contribute to backend API development for a ride-sharing aggregation platform. Previously, I worked as a full stack developer at System Soft Technologies building client-facing web applications with Node.js and React.

## What I'm building

**[AgentMeld](https://agentmeld.com)** is a service I run that bridges asynchronous cloud agents with local coding agents, giving developers a way to offload long-running agentic tasks to the cloud while keeping local context. It's early-stage and actively in development.

I also develop **abilities for [OpenHome](https://openhome.ai)**, a voice AI and smart speaker platform. The OpenHome team selected me for their developer grant program and provided a hardware dev kit to build on.

In 2024, I founded **Kairobo Platforms LLC** to explore low-cost autonomous ground vehicles using consumer mobile devices as the compute platform. After validating the technical feasibility and scope, I dissolved the company pre-product.

## What drives me

I believe in learning by building. Whether it's a Rust-based CLI tool, an Arduino library that controls a Roomba, or an MCP server that wakes up a PS5 — every project is a chance to go deeper into a technology and ship something real.
