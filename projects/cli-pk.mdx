A focused, AI-powered coding assistant with support for Anthropic Claude and OpenRouter models. CLI-PK helps you write, analyze, and execute code directly from your terminal.

## What it does

CLI-PK is a terminal-based coding assistant that connects to multiple LLM providers. It gives you a conversational coding workflow without leaving the command line.

## Highlights

- Multi-provider support — Anthropic Claude and OpenRouter models
- Code generation, analysis, and execution from the terminal
- Built with JavaScript for broad compatibility
