A Model Context Protocol (MCP) server that enables Claude Code to discover, register, and wake PS5 consoles on your local network.

## What it does

This MCP server bridges Claude Code and your PS5. It uses network discovery and Wake-on-LAN to let you find and power on your console through natural language in the terminal.

## Highlights

- MCP server built with TypeScript
- PS5 network discovery via local network scanning
- Wake-on-LAN support to power on consoles remotely
- Integrates directly with Claude Code's tool system
